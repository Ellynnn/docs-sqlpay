This release adds **CIMB Bank Bulk DuitNow** file format, prepends month and year to the **RHB Reflex EPF** payment filename, optimizes TMS sync performance, and resolves multiple tax form and bank file issues. (*239*)

<!-- truncate -->

## Features

- Added **CIMB Bank Bulk DuitNow** file format (#6294)
- Added month and year prefix to the **RHB Reflex EPF** payment text filename (#6304)

---

## Updates

- Optimized **Sync Cloud TMS** performance by limiting data query to 6 months
- Allowed viewing **Time Attendance Summary** and **Unassigned Listing** reports older than 6 months even when incomplete logs exist

---

## Bug Fixes

- Fixed incorrect bank code in **Al-Rajhi Bank** Payment Upload Excel format (#6311)
- Fixed **PCB** amount populated in the wrong column in the **CP22A** file (#6307)
- Fixed Spouse **Passport** value populated in the wrong column in the **CP22** file (#6305)
- Fixed mismatch in compensation and gratuity amounts between **Income Tax EA** form and detail report (#6303)
