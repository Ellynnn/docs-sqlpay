This release improves time attendance log editing with warning prompts, editing restrictions on **Leave/OT** records, and fixes an incorrect clock status display after resolving an error log. (*239*)

<!-- truncate -->

## Updates

- Revised warning prompt when editing a log entry for yesterday's date
- Disabled editing of **Leave/OT** records when the transaction date is yesterday or later
- Added validation requiring **Clock In** time to be entered before editing **Clock Out** time

---

## Bug Fixes

- Fixed **Clock Status** incorrectly showing Absent/No Clock Out when the log is complete after resolving an error log
